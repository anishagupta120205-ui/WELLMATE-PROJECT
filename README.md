# 🧠 WellMate – AI Driven Student Well-Being Assessment, Distress Detection and Recommendation Platform

## 📌 Project Overview

WellMate is an AI-powered Student Wellness Assessment and Recommendation Platform designed to monitor, analyze, predict, and improve student well-being using Machine Learning, Behavioral Analytics, Predictive Modeling, and Personalized AI-driven interventions.

The platform integrates academic behavior, digital activity patterns, lifestyle habits, wellness indicators, and psychological factors to assess student wellness status, predict stress and burnout risk, detect early distress signals, and provide personalized recovery recommendations.

Unlike traditional wellness applications that depend mainly on self-reported surveys or generic wellness advice, WellMate follows a proactive and predictive AI-driven approach by continuously analyzing behavioral patterns and identifying early warning signs of declining student well-being.

---

## 🎯 Problem Statement

Students face increasing academic pressure, examination stress, project deadlines, internships, placements, poor sleep cycles, excessive screen time, social media overuse, digital fatigue, burnout, and unhealthy lifestyle habits.

Most existing wellness systems focus on isolated interventions such as mood tracking, meditation applications, counseling services, or generic chatbot support.

Current limitations include:

* Dependence on self-reported surveys
* Lack of continuous behavioral monitoring
* Generic recommendations for all users
* Absence of predictive wellness assessment
* Lack of early burnout detection mechanisms
* Limited integration of academic and lifestyle factors

WellMate addresses these gaps by integrating Artificial Intelligence, Machine Learning, Behavioral Analytics, and Predictive Wellness Assessment into a unified student wellness ecosystem.

* Sleep Duration

* Screen Time

* Wellness Index
* Stress Risk Score

Models implemented:

* Logistic Regression
* Naive Bayes
* Decision Tree Classifier
* Random Forest Classifier
* Support Vector Machine (SVM)
* K-Nearest Neighbors (KNN)

Best Model Selected:

**Random Forest Classifier**

Reason:

* High accuracy
* Better non-linear pattern learning
* Reduced overfitting through ensemble learning
